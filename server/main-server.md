## Информация о cервере

**Параметры**: 1 ядро процессора, 2 Гб оперативной памяти, 40 Гб диска, 1 IP-адрес

**Операционная система**: Ubuntu 20.04

**IP-адрес сервера**: 188.120.225.37

**Установленное ПО**:

- Nginx
- MySQL
- PHP
- Wordpress
